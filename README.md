# Secure-Departmental-Network-Architecture

Project Overview: Designed and simulated a secure departmental network using Cisco Packet Tracer to enable communication between university departments. The project focuses on implementing routing, IP addressing, and a scalable network architecture to ensure reliable and efficient data exchange.

Tools Used: Cisco Packet Tracer IP Addressing & Routing Configuration

Network Components: Routers, Switches, PCs, Access Points, DNS Server

Key Configurations: IP addressing scheme implementation, Router and switch configuration, Routing table setup, Inter-department connectivity testing

Objectives: Enable secure communication between departments, Improve network reliability and scalability, Simulate a real world campus network environment
